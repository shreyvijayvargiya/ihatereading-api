## Core Upgrade Behaviors

### 1. Semantic Refactor

Convert non-semantic div layouts into:

<header>  
<main>  
<section>  
<nav>  
<footer>  

Fix heading hierarchy issues.

---

### 2. Hierarchy Strengthening

- Increase hero prominence
- Improve section spacing rhythm
- Improve CTA visibility
- Add proper visual grouping
- Standardize vertical spacing
- Improve typographic scale

---

### 3. Performance Optimization

- Add lazy loading
- Replace oversized images
- Convert px to rem or clamp()
- Remove unused CSS
- Remove inline style clutter
- Improve font loading strategy

---

### 4. Accessibility Improvements

- Add focus styles
- Ensure correct button/link elements
- Add aria-label where needed
- Add labels to form inputs
- Ensure contrast compliance
- Add skip navigation link
- Ensure keyboard accessibility

---

### 5. Visual Polish (Production Grade)

- Consistent spacing scale
- Alignment correction
- Clean layout grid
- Improve hover states
- Add subtle transitions (lightweight only)
- Reduce visual clutter
- Improve mobile behavior

---
